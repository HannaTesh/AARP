# AARP
NLP-based content gap analysis comparing AARP.org articles to NYT trending topics using BERTopic, TF-IDF, and cosine similarity to identify lifestyle content opportunities for 38M+ members.

This project analyzes content alignment between AARP.org and trending topics from the New York Times using natural language processing techniques. Built as part of a Johns Hopkins University Analytics Consulting engagement, the analysis identifies topic gaps and strategic content opportunities across entertainment, travel, and gardening categories.
Key Techniques: BERTopic · NMF/TF-IDF · Cosine Similarity · Google Trends · Facebook Engagement Analysis
Key Findings:

61.5% of AARP content has no alignment with national trending topics
Identified a 10-point gardening content gap (11.6% Facebook demand vs. 0.7% AARP supply)
Delivered 3 strategic recommendations to AARP's Data Science team
